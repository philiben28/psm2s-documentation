# PSM2S - VISION PRODUIT

Version : 1.0

---

# Notre mission

PSM2S est une plateforme de pilotage de la conformité des établissements.

Notre objectif n'est pas de remplacer un registre papier.

Notre objectif est d'aider les responsables à anticiper, organiser et suivre toutes leurs obligations réglementaires.

---

# Notre vision

Le Directeur doit pouvoir ouvrir PSM2S chaque matin et comprendre en moins de 30 secondes :

- ce qui est conforme ;
- ce qui est en retard ;
- ce qui arrive prochainement ;
- les actions qu'il doit réaliser.

---

# Notre philosophie

La simplicité est prioritaire.

L'utilisateur ne doit jamais chercher une information importante.

Le logiciel doit la présenter automatiquement.

Chaque évolution doit faire gagner du temps.

---

# Les quatre piliers de PSM2S

## 1. Anticiper

Prévenir avant qu'un problème n'apparaisse.

Exemples :

- calendrier réglementaire ;
- alertes ;
- notifications.

---

## 2. Piloter

Donner une vision immédiate de la situation.

Exemples :

- tableau de bord ;
- widgets ;
- score de conformité.

---

## 3. Prouver

Conserver toutes les preuves de conformité.

Exemples :

- rapports ;
- historiques ;
- pièces jointes ;
- registres.

---

## 4. Automatiser

Éviter les tâches répétitives.

Exemples :

- calcul automatique des échéances ;
- génération de documents ;
- rappels automatiques.

---

# Les principes de développement

Une donnée ne doit être saisie qu'une seule fois.

Le logiciel doit réutiliser les informations déjà présentes.

Une fonctionnalité ne doit jamais compliquer le travail de l'utilisateur.

---

# Priorité actuelle

La priorité stratégique est le développement du **Calendrier Réglementaire Intelligent**.

Ce calendrier constitue le cœur du pilotage de PSM2S.

Toutes les évolutions futures devront pouvoir s'y intégrer.

---

# Notre règle d'or

Chaque nouvelle fonctionnalité doit rendre PSM2S plus indispensable, pas simplement plus complet.

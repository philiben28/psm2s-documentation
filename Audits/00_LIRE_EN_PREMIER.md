# Présentation du projet

PSM2S est une plateforme de pilotage des obligations réglementaires destinée aux établissements recevant du public.

L'objectif n'est pas seulement de gérer des registres, mais de fournir un véritable outil d'aide au pilotage permettant de :

- centraliser les obligations réglementaires ;
- suivre les contrôles périodiques ;
- anticiper les échéances ;
- produire les documents réglementaires ;
- accompagner les responsables d'établissement dans leur conformité.

La philosophie du projet privilégie :

- simplicité d'utilisation ;
- fiabilité ;
- sécurité ;
- évolutivité ;
- maintenabilité.

# Vision du projet

PSM2S évolue progressivement d'un logiciel de gestion de registres vers une plateforme complète de pilotage réglementaire.

Les futures évolutions pourront notamment intégrer :

- tableaux de bord ;
- calendrier réglementaire ;
- alertes intelligentes ;
- DUERP ;
- PCA/PRA ;
- Accessibilité ;
- CyberScore ;
- automatisation documentaire ;
- assistance par IA.

Chaque évolution devra renforcer la valeur apportée aux établissements sans remettre en cause la stabilité du socle existant.
